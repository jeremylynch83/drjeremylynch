# Zanda booking update notes

This update improves the appearance of the enquiry modal while keeping the existing working behaviour.

Changes:

- modal width reduced to 560px
- modal panel changed to a softer transparent black style
- modal border softened
- form fields made slimmer
- Send and Cancel buttons restyled to match the website buttons
- Send and Cancel buttons sit side-by-side on desktop and stack on mobile
- more spacing added between the modal heading and form
- page background now uses a darker translucent overlay with a subtle blur
